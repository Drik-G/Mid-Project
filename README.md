=== SPACE SHIP SIMULATION ===
Running simulations of two space ships with assigned names, fuel and integrity level.

=== HOW TO ===
1. Open terminal in the source folder.
2. Compile the project by typing: javac Main.java
3. Run the simulation by typing: java Main

=== REFERENCE ===
Encapsulation: in SpaceShip.java, variables are private.
Inheritance: ReconShip.java and CargoShip.java extend the SpaceShip.java superclass.
Overriding: Both child classes override the method in the parent class .PerformMission() (runtime polymorphism)
overloading: an overloaded .PerformMIssion(int lightYears) method in SpaceShip.Java
